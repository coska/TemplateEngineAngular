# Command Line Tools

- new post

```bash
npm run cli:post <menu_id>/<sub_menu_id>/<post_id>
```

```bash
npm run cli:post ./src/assets/section/m03/m030103/
npm run cli:post ./src/assets/section/m03/m030103/
npm run cli:post m03/m030103/howto-crate-a-post

npm run cli:post m03/m030103

```
